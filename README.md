# 💁 Introduce
* FPGA(Cora_z7) 보드를 이용 FND를 사용하여 시계를 구현
* Verilog언어를 Vivado 프로그램을 사용하여 설계

# 📀 Basic Features
* 🕐 시계(셋팅모드 포함)
* ⏱️ 스톱워치(lap 기능 포함)
* ⏲️ 타이머(알람기능 포함)

# 🔎 Detailed features
